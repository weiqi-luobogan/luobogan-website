# 电子签约工具对比与开户指引

> **文件说明**：本文档为围棋启蒙机构选择电子签约工具的落地参考。已按"家长侧走法大大、教师内部走 e 签宝或纸质、加盟走法大大"的策略给出主推方案，并附开户流程与家长端签署体验说明。
> **使用范围**：机构管理层与校长在正式启用电子签约前，用作选型与培训手册。
> **起草日期**：2026-07-16　**版本**：v1
> **法律依据**：《中华人民共和国电子签名法》第 3 条、第 13 条、第 14 条明确"可靠的电子签名"与手写签名或盖章具有同等法律效力，前提是签署人身份可识别、签署意愿真实、内容不可篡改、有可追溯的存证。

## 二、法大大（主推）——开户与使用流程

### 2.1 适用场景

- 家长侧《围棋启蒙培训服务合同》《入学知情同意与免责声明》《退费与转班规则协议》；
- 加盟侧《加盟合作框架协议》及各项附件。

### 2.2 为什么优先选它

1. 存证与司法链路完备：区块链存证 + 可信时间戳 + 与多家公证处、司法鉴定所、互联网法院对接，发生争议时可一站式出具证据。
2. 家长端体验轻量：家长通过短信或微信收到签署链接，H5 页面即可完成实名 + 人脸认证 + 签署，全程 2–3 分钟。
3. 累计签署量在国内头部（官方公示 122 亿份），品牌可信度高，可作机构合规背书之一。
4. 按次套餐灵活，前期无需一次性购买大量席位。

### 2.3 开户流程（企业主体）

1. **注册企业账号**
   - 前往 https://www.fadada.com/ 点击"注册"，选择"企业用户"；
   - 填写机构名称、统一社会信用代码、法人姓名、联系人、联系电话。
2. **企业实名认证**
   - 上传营业执照原件或彩色扫描件；
   - 法人身份证正反面扫描 + 法人人脸识别（可远程视频认证）；
   - 提供机构对公账户信息用于打款验证（一般 1–2 个工作日完成）。
3. **购买套餐**
   - 家长侧可先购买 500 份／1000 份的按次套餐；
   - 后续按招生进度扩容。
4. **模板上传**
   - 将 6 份模板（.docx 或 .pdf）上传后台；
   - 在模板中拖拽定位签署位、日期位、指定字段（如姓名、身份证号）；
   - 保存为"合同模板"，签约时通过"发起签署"选择模板并填入字段值。
5. **签署印章设置**
   - 上传机构公章或电子印章文件（无电子印章可用系统自动生成的电子公章）；
   - 设置法定代表人或授权代表的电子签名。

### 2.4 家长端签署体验

1. 校长／老师在后台"发起签署"，填入家长姓名、手机号、身份证号、合同关键字段；
2. 系统自动向家长发送短信 + 微信通知；
3. 家长点击链接进入 H5 页面 → 人脸识别 → 阅读合同 → 手写签名 → 提交；
4. 双方签署完成后，系统自动生成含区块链存证信息的 PDF 合同副本，双方均可下载。

### 2.5 存证与下载

- 每份合同签署完成后，系统同步生成《存证证书》，包含 Hash 值、时间戳、区块链上链信息；
- 发生纠纷时可申请"司法鉴定意见书"，作为诉讼证据；
- 建议每月将存证证书打包下载并同步至本地备份盘。

## 三、e 签宝（备选）——开户与使用流程

### 3.1 适用场景

- 教师内部合同（全职、兼职）；
- 需要通过支付宝端内快速调起签署的临时场景；
- 与家长中已高度依赖支付宝生态的家庭签约。

### 3.2 与法大大对比要点

| 对比项 | 法大大 | e 签宝 |
| :--- | :--- | :--- |
| 生态入口 | 短信 + 微信 H5 + 独立 APP | 支付宝小程序 + 微信 + 独立 APP |
| 家长端习惯 | 依赖短信/微信触达 | 家长打开支付宝直接搜"e 签宝" |
| 教师内部签约体验 | 支持员工内部签署 | 支付宝端一键实名，教师少走一步下载 |
| 单份成本 | 3–6 元 | 3–5 元 |
| 与法大大差异 | 存证 + 司法链路更强 | 与支付宝、蚂蚁链协同更紧 |

### 3.3 开户流程（企业主体）

1. 访问 http://s.shuziqianzhan.com/esign 或搜索"e 签宝"官网点击"注册"；
2. 选择"企业实名"，上传营业执照、法人身份证、对公账户信息；
3. 完成企业与法人的实名 + 授权（可通过支付宝企业实名快速互认）；
4. 上传教师侧模板（全职、兼职），拖拽签署位；
5. 购买按次套餐或年度套餐；
6. 教师入职时通过短信或支付宝端签署即可。

### 3.4 教师内部签署体验

1. HR 在后台发起签署 → 教师收到短信；
2. 教师打开支付宝 → 首页搜索"e 签宝"→ 一键实名（若教师已完成支付宝实名，可跳过）；
3. 阅读合同 → 手写签名 → 提交；
4. HR 侧一键盖公章 → 签署完成 → 双方下载合同与存证。

## 四、不推荐工具说明

### 4.1 DocuSign

- 全球总部在美国，服务器境外；
- 校外培训涉及大量学员未成年人个人信息、家长身份证信息、健康信息等，交由境外服务器存储存在数据主权与合规风险；
- 中文本地化体验相对一般，家长端不熟悉；
- **结论**：不推荐用于家长／学员数据相关的签约。

### 4.2 契约锁 / 上上签

- 主要面向中大型企业客户，报价与最小起订量偏高；
- 中小型校外培训机构使用成本相对不划算；
- 对家长个人签署的轻量化触达设计不如法大大、e 签宝；
- **结论**：中小型围棋机构初期不推荐，除非未来规模化后有集中合同管理需求。

## 五、机构落地建议

### 5.1 分场景分工

| 签约场景 | 推荐工具 | 说明 |
| :--- | :--- | :--- |
| 家长侧培训服务合同 + 知情同意 + 退费协议 | 法大大 | 存证与司法链路更完备 |
| 全职教师聘用合同 | e 签宝或纸质签署 | 教师群体使用支付宝实名率高 |
| 兼职教师课时费结算协议 | e 签宝 | 简单快速，短期合作场景合适 |
| 加盟合作框架协议 | 法大大 | 大额合同 + 品牌方证据链需求 |

### 5.2 上线动作清单

- ☐ 完成企业实名（法大大、e 签宝各一次）；
- ☐ 6 份模板全部上传后台并预置签署位；
- ☐ 电子公章、法人签名或授权代表签名上传完毕；
- ☐ 招生顾问、班主任、HR、加盟经理 4 岗位培训到位；
- ☐ 家长／教师／加盟商签约操作指引（≤ 1 页）已同步给相关岗位；
- ☐ 每月存证证书打包下载并本地备份的机制已落实；
- ☐ 与保险公司、法务顾问约定：发生争议时电子存证提取与司法鉴定的对接路径。

### 5.3 常见问题

1. **家长完全不会用智能手机怎么办？**
   - 保留纸质签署通道；纸质签署完成后由校区扫描存档，与电子合同同等归档管理。
2. **签署过程中家长中途退出如何处理？**
   - 系统会生成"未完成"记录；招生顾问可 24 小时内再次发起，避免重复计费。
3. **合同签署完成后需要长期保存吗？**
   - 建议至少保存至学员离校后 10 年（与财务凭证保存年限匹配）；电子存证平台默认长期在线可查。
4. **家长要求纸质合同副本怎么办？**
   - 从电子签约平台下载完整 PDF（含存证信息），打印装订后加盖机构合同专用章交付家长即可。
