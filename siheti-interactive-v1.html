CORE FRAMEWORK · 三维

眼位 · 缺陷 · 棋形

死活题的三根主脊。切换页签，逐一看。

目
EYE SPACE

看透一块棋能不能做出两只真眼

眼位是死活的落脚点。判断一块棋，先看它有几只已成的眼、几只可做的眼、有没有分割成两只眼的空间与要点。眼位问题分两条大线：形状要点 与 空间大小。

01
常见形状要点
虎口、丁四、刀五、直三——每一种都对应固定的急所与做活方向，需成为条件反射。
02
角上常见形状
板六、板八在角部有特殊活死规律，与边上不同。角上眼位受二线一线气紧影响，判断规则要单列。
03
眼位空间大小
扩大眼位是做活的第一直觉，缩小眼位是杀棋的第一直觉。空间越小、越接近死形。
04
眼位形状问题
占据要点让形状活起来；找到分割点让整块棋一分为二。眼位问题最终收敛到要点与分割。
隙
FLAWS

断点与气紧，是所有死活题的隐藏抓手

棋形上的缺陷，一端是自己的软肋，一端是敌人的软肋。做活要补自己的断点，杀棋要抓对方的断点。缺陷训练强化的是空间感：看到形状就能感到哪里紧、哪里薄。

01
断点
哪里能被切开、切开后局部气还剩几口。断点是杀棋的入口，也是做活必须优先补掉的伤口。
02
气紧
一块棋能承受几步进攻。气紧的一方回旋余地小，往往被"先手叫吃"逼出问题手。
03
应对与利用
对自身缺陷——先补好再谈做活；对对方缺陷——抓住它当作破眼、劫材、制造打劫的抓手。
04
空间感认知
训练久了，看到一块棋，气与断点会像光斑一样自动浮现——这就是老师说的"棋感"。
形
SHAPE

从"看着舒服"到"算得清方向"

棋感的本质是大量常形在脑海里形成图片记忆。棋形对了，方向就对了；方向对了，计算才是有意义的。棋形训练是把"直觉"变成"可解释的直觉"。

01
简单棋感形成
见得多，才会条件反射。常形记忆库越大，看棋越快，负担越小。
02
有方向性的计算
先靠棋感锁定 2-3 个候选点，再深入算，避免陷入"每一步都算三十步"的黑洞。
03
思维图片能力
脑内能"看见"棋盘变化。这是做活/杀棋能否精准落子的底层能力。
04
棋理理解
知其然更知其所以然。理解"为什么这里是急所"，才能把一题变成一类。
METHOD · 三步

观察 → 计算 → 验证

死活题不是撞出来的，是走出来的。 三步法，把一次解题拆成看得见的动作。

STEP · 01
观察

先别急着算。把这块棋看成一张地图，找到出发点。

点击翻面 →
看什么
  • 判断任务：做活，还是杀棋？
  • 数眼位空间：有几只成眼、几只可做
  • 数气：自己几口气、对方几口气
  • 找急所：形状的中心 / 分割点在哪
STEP · 02
计算

锁定候选点，逐一往下推。双方走最强手，看能推到哪一步。

点击翻面 →
怎么算
  • 推演双方的最强应对
  • 穷尽变化路径，不能只看一条
  • 算到局部收官为止
  • 结论落到：净活 / 净死 / 打劫
STEP · 03
验证

正解不是终点。回过头再看一遍，把细节和棋理都拧紧。

点击翻面 →
验什么
  • 正解里有没有被忽略的应对
  • 失败线里，是哪一步走偏了
  • 这一题背后是哪条棋理
  • 能不能迁移到同类形状

MANTRAS · 口诀

四死六活 · 五补七补 · 六死八活

口诀是老师这些年在课上讲了几百遍、孩子们在题里遇了几千遍以后， 浓缩下来的位置换算表——记牢，能省一半计算。

CORNER · 角部
四死六活
角上四个交叉点做不出两只眼；六个交叉点才有活的空间。
SIDE · 边部
六死八活
边上六个交叉点是死形；八个才够转身做出两只真眼。

五补七补
五、七两档是临界值——落子补一手，活；不补一手，死。

TACTICS · 战术库

做活五招 · 杀棋三招

任何一道死活题的正解，都在这八招之内。 点开每一招看：原理 · 关键要点 · 核心洞察。

1
扩大眼位立 / 扳 / 虎 / 跳
▾
PRINCIPLE · 原理

眼位空间越大，做出两只真眼的可能就越大。所以做活的第一直觉，是"往外伸一点"。

KEY POINTS · 关键要点
  • 立：向自己方向立一手，把眼位向纵深延伸
  • 扳：从边上顶一手，把敌人的压缩顶回去
  • 虎：形成虎口，同时留住眼位和气
  • 跳：一间跳出去，扩大势力范围
核心洞察：扩大不是随便扩，扩到能同时容纳两只真眼的空间，才叫扩对了。
2
占据要点棋形中心 · 急所
▾
PRINCIPLE · 原理

每种常形都有一个"急所"——那一手落下去，形状就活；不落，就死。要点是形状的心脏。

KEY POINTS · 关键要点
  • 找到形状中心：多数急所在几何中心附近
  • 先手抢占：谁先占，谁就把主动权拿走
  • 被抢先了，就要看能不能靠"分割"或"扩大"补救
核心洞察：做活与杀棋的急所往往是同一个点——所以这一点，谁走都是好棋。
3
分割大眼化整为零
▾
PRINCIPLE · 原理

一整块大空间往往只能做出一只眼。找准分割点，把它切成两块小的，才能各出一只眼。

KEY POINTS · 关键要点
  • 识别"看着大、其实一只眼"的假象
  • 分割点：把大空间从中间隔出两片
  • 分割之后，两片各自都要有做眼空间
核心洞察：一分为二，是把"貌似做活"变成"真的做活"的关键动作。
4
利用缺陷制造打劫扑 / 挖
▾
PRINCIPLE · 原理

净死之前，先看看能不能拖到打劫。打劫等于"给自己留一个复活的机会"。

KEY POINTS · 关键要点
  • 扑：往对方虎口里送一子，制造劫材
  • 挖：从对方的断点插进去，逼出打劫形
  • 先看清劫的性质：本身劫 / 缓一气劫 / 万年劫
核心洞察：打劫是死活里的"下策做活"——够不上净活，就退一步；但要清楚劫材够不够用。
5
特殊战术胀牯牛 · 三眼两做
▾
PRINCIPLE · 原理

常规四招都不成时，形状本身可能带着"隐藏机关"——胀牯牛、三眼两做，都是不按常理出牌的诡招。

KEY POINTS · 关键要点
  • 胀牯牛：利用气紧，让对方送吃反成活棋
  • 三眼两做：三个可做眼位，敌人堵一处，就在另一处做
  • 此类招式对棋感依赖极高，需要日积月累积累到形状库
核心洞察：特殊战术不是"多招"，是"你先熟悉了形状，形状才愿意告诉你这条秘密路线"。
1
缩小眼位扳 / 点 / 扑 · 从外围压缩
▾
PRINCIPLE · 原理

做活先扩眼位，杀棋反着来——把对方的眼位空间一点点往回压，压到做不出两只眼为止。

KEY POINTS · 关键要点
  • 扳：从外围顶一手，让对方眼位少一格
  • 点：进到对方最外一线，堵住扩眼路径
  • 扑：往对方虎口送吃，切断做眼空间
核心洞察：缩小眼位的目标不是"压死"，是"压到只剩一只眼"——只要少一只眼，就已经死了。
2
点眼一点死 · 刀五 / 直三
▾
PRINCIPLE · 原理

某些棋形本质是"假活"，急所被点一手，眼位当场归零。这一手叫"点眼"。

KEY POINTS · 关键要点
  • "一点死"棋形：刀五、直三、丁四、方四 等
  • 点眼的落子位置：形状的几何中心 / 眼位急所
  • 被点之后即使敌方补一手，也无法回到两眼形
核心洞察：点眼靠的是形状识别——认得住形，一眼就看到那一点在哪。
3
利用缺陷挖 / 断 / 夹 / 扑 · 棋从断处生
▾
PRINCIPLE · 原理

对方棋形有断点、有气紧，就有生路。把这些缺陷抓住，靠"挖断夹扑"把整块棋打散。

KEY POINTS · 关键要点
  • 挖：插入敌方断点，切断联络
  • 断：把两块棋切开，各自算气
  • 夹：从两面夹击一子，逼对方走软
  • 扑：先送一子，利用叫吃扭转局面
核心洞察："棋从断处生"——所有精妙的杀棋，几乎都是从对方那个不起眼的断点开始的。

TRAINING · 训练

走对训练路径

死活题不是刷出来的，是用对方式练出来的。 先看清常见误区，再对照正确路径走。

COMMON PITFALLS · 常见误区

这样练，越练越糊

  • 只做题，不实战——题上会，棋盘上还是发懵
  • 把"做题"当作围棋的全部，忽略实战里的效率与子效
  • 实战中主动走成死活题：局部被算准，反倒被吃
  • 只求答案，不复盘：为什么这一手对？为什么那一手错？说不出来
  • 题目背下来了，形状换个方向就不认识
RIGHT WAY · 正确路径

这样练，能落到实战

  • 训练目标要清楚：思维图片能力、棋感、计算逻辑、常形记忆
  • 刷题后必复盘：讲得出棋理，才算真的会
  • 常形记忆库要有意识地积累：见一次记一次
  • 题上练精算，棋上练判断——两条腿走路
  • 实战中主动避免走成"死活题局面"，才是高子效的下法
