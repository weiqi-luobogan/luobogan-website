# 退费与转班规则协议

> **文件说明**：本协议为《围棋启蒙培训服务合同》的必备附件，用于对退费、转班、停课、赠课等常见场景做细化约定，与主合同表述保持一致，如与主合同发生冲突以主合同和主管部门规定为准。
> **使用范围**：所有报名甲方线下／线上围棋启蒙、进阶课程的学员及其法定监护人须签署。
> **起草日期**：2026-07-16　**版本**：v1
> **底线原则**：任何退费规则均须遵守单次预付"3 个月／60 课时／5000 元"上限、监管账户"一课一销"划拨、"未消课时全额退还"这三条硬性底线；违反上述底线的条款一律无效。

## 一、通用定义

（一）"已消课时"指学员实际到课并接受教学的课时，以及乙方无故缺席且未在合同约定时限内请假的课时。

（二）"未消课时"指本合同项下已缴纳培训费但尚未按上述规则计为已消的课时。

（三）"合同单价"指本合同项下的单课时价格，即 {单价} 元／课时；如因促销、组合套餐等因素存在多种价格，按"实际支付培训费 ÷ 总课时数"计算加权单价。

（四）"赠送课时"指甲方通过"老带新推荐福利、老学员续费福利、活动福利、生日福利"等方式赠送的课时，本身不单独作价。

## 二、常见退费与转班场景一览

| 场景 | 触发条件 | 退费金额计算 | 违约金 | 到账时限 |
| :--: | :--- | :--- | :--: | :--: |
| 未开课全额退 | 培训未开始，乙方书面申请 | 全额退还已付培训费 | 无 | ≤ 20 个工作日 |
| 正常按已消课时退 | 已开课，双方书面确认解除 | 已消课时按合同单价扣除，未消课时全额退还 | 无 | ≤ 20 个工作日 |
| 因个人原因退费 | 乙方搬迁、学员健康、家庭安排等 | 已消课时按合同单价扣除，未消课时全额退还，赠送课时按第五条处理 | 无 | ≤ 20 个工作日 |
| 因机构原因退费 | 师资、场地、课程质量与约定严重不符或甲方停业 | 全额退还未消课时费用，并按剩余培训费 {甲方违约金比例}% 支付违约金 | 有 | ≤ 20 个工作日 |
| 转班（跨班次） | 乙方申请更换班级／校区／时段 | 按合同单价折算已消课时，剩余部分转入新班；单价差异部分多退少补 | 无 | 即时办理 |
| 停课保留 | 乙方申请短期停课 | 未消课时账户保留，不结算 | 无 | 保留期见第四条 |
| 老带新赠课 | 老学员推荐新学员成功报名 | 按第五条约定赠送课时处理 | 无 | 见第五条 |
| 不可抗力停课 | 战争、自然灾害、传染病、政府命令 | 顺延课时或按已消课时结算 | 无 | ≤ 20 个工作日 |

## 三、退费办理流程

（一）乙方发起

1. 通过"校外培训家长端"APP 发起线上退费申请；或
2. 通过校区前台、班主任书面提交《退费申请表》，注明学员姓名、班次、退费原因、剩余课时、退款账户信息。

（二）甲方审核

1. 甲方在收到申请后 {退费审核工作日数}（≤ 3）个工作日内完成审核，出具书面《退费确认单》，双方签字后作为退费凭证。
2. 审核期间甲方不得中止乙方尚未申请退费的课程。

（三）资金划拨

1. 走监管账户资金按"未消课时对应培训费 × 加权单价"计算；
2. 退款一律原路返回至乙方缴费账户；
3. 退款到账时限不超过 20 个工作日；超期部分甲方按逾期金额 {逾期违约金日比例}%（不超过万分之六点五）每日向乙方支付违约金。

（四）票据与凭证

1. 已开具发票的，乙方须配合退回或提供发票冲红所需信息；
2. 退费完成后甲方保留 {票据留存年限} 年退费档案备查。

## 四、转班、停课、复课规则

（一）转班

1. 乙方可申请在甲方旗下不同校区、不同班次、不同时段、不同班型之间转班。
2. 转班后按新班次的公示单价结算：新单价高于原单价的，差额部分由乙方补足；新单价低于原单价的，差额部分退还乙方或转为账户余额。
3. 因甲方原因导致的转班（撤班、师资调整等），不得单方面提高单价。

（二）停课保留

1. 乙方可因短期健康、外出、家庭安排等原因申请停课；一次连续停课时长上限为 {单次停课上限} 个月，累计停课不超过 {累计停课上限} 个月。
2. 停课期内课时账户保留，甲方不得单方面注销；
3. 因不可抗力导致的停课，不计入上述上限。

（三）复课

停课到期后学员未办理复课或退费手续，且甲方通过登记电话／微信提示 {失联提示次数} 次以上仍无回应的，甲方可将账户转入"待处理"状态，但账户内未消课时对应培训费仍归乙方所有，乙方随时可以要求继续消课或办理退费。

## 五、赠送课时处理

（一）赠送课时来源

1. 老带新推荐福利：老学员推荐新学员成功报名并完成 {老带新达标课时} 节以上正常上课的，甲方赠送老学员 {老带新赠课节数} 节课时。
2. 活动福利：节日活动、机构周年庆、赛事福利等。
3. 续费福利：老学员一次性缴费达到 {续费达标金额} 元并落入"365"上限内的，甲方赠送 {续费赠课节数} 节课时。
4. 其他甲方自愿赠送的课时。

（二）赠送课时使用

1. 赠送课时在正式课时全部消耗后按顺序使用，或按甲方与乙方另行约定使用；
2. 赠送课时不作价、不折现、不单独退费；
3. 乙方申请退费时，如已按顺序消耗一部分赠送课时，剩余的赠送课时同时作废，不冲抵实际退费金额；
4. 因甲方原因导致合同无法履行的，乙方账户中的赠送课时可申请一次性折算为等值课时补偿（按合同单价计算），也可选择放弃。

## 六、双方特别声明

（一）本协议中所有退费金额均以监管账户内实际留存资金为准；甲方不得以"课时账户余额已结转、赠送课时已抵扣、经营困难、账户暂时资金紧张"为由拖延、拒绝退费。

（二）本协议条款不免除甲方因自身故意或重大过失导致的责任；如与法律法规或主管部门规定冲突，以后者为准。

（三）双方就退费与转班事项产生争议的，按主合同《围棋启蒙培训服务合同》第七条约定处理。

## 七、生效与签署

本协议自甲乙双方签字或电子签署之日起生效，与主合同同时执行；主合同终止时，本协议亦终止，但已发生的退费义务不因终止而免除。
