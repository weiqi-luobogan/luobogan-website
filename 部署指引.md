## 第二步｜准备 GitHub 仓库

**注册 GitHub**：https://github.com/signup

**新建仓库**：
1. 登录后右上角 `+` → `New repository`
2. Repository name 填：`luobogan-website`
3. 勾选 `Public`
4. 点 `Create repository`

**上传文件**：
1. 仓库页点 `uploading an existing file`
2. 把 `website_v0/` 目录里的 **3 个文件夹/文件全部拖进去**：
   - `index.html`
   - `assets/`（整个文件夹）
3. 下方 `Commit changes` 提交

---

## 第三步｜Vercel 部署

**注册 Vercel**：https://vercel.com/signup
- 选「Continue with GitHub」用 GitHub 账号一键登录

**部署网站**：
1. 登录后点 `Add New...` → `Project`
2. 找到刚才的 `luobogan-website` 仓库，点 `Import`
3. Framework Preset 选 `Other`，其他全部默认
4. 点 `Deploy`
5. 等 30 秒，看到 🎉 即上线

**临时访问地址**：
- Vercel 会给你一个 `luobogan-website-xxx.vercel.app` 临时域名
- 用这个先确认网站没问题

---

## 第四步｜绑定自有域名

**Vercel 侧**：
1. 项目页点 `Settings` → `Domains`
2. 输入 `weiqiluobogan.com`，点 `Add`
3. 再输入 `www.weiqiluobogan.com`，点 `Add`
4. Vercel 会给你 2 条 DNS 配置指引（A 记录 + CNAME）

**域名注册商侧**（阿里云/Cloudflare 后台）：
1. 进入域名解析管理
2. 按 Vercel 给的指引添加 2 条记录：
   - `@` → A 记录 → `76.76.21.21`
   - `www` → CNAME → `cname.vercel-dns.com`
3. 保存

**等 5-10 分钟 DNS 生效** → 浏览器打开 `weiqiluobogan.com`，看到网站即成功 ✅
